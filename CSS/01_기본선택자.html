CSS 기본 선택자

선택자란?

특정 html요소를 선택하고자 할때 사용하는 기능
해당 요소를 선택해서 원하는 스타일과 기능을 적용시킬 수 있다.

1.모든(전체) 선택자 (*)

모든 요소를 선택하고자 할때 사용 

        *{
            스타일 속성 : 값;
            스타일 속성 : 값;
        }

2.태그 선택자 (태그명)

        태그명{
            스타일 속성 : 값;
        }

3.아이디 선택자(#아이디명)

해당 문서내에 특정 html요소 하나만을 선택할때 사용
단 해당 요소 id속성을 이용해서 고유 id값을 부여해야한다. 

        #아이디명{
            스타일 속성 : 값;
        }
  1. 아이디 선택자
  2. 아이디 선택자
  3. 아이디 선택자
  4. 아이디 선택자

4.클래스 선택자(.클래스명)

해당 문서내에 내가 원하는 여러개의 요소를 선택하고자 할때 사용
클래스 속성을 추가해야한다. 

        .클래스명{
            스타일 속성 : 값;
        }
  1. 클래스선택자1
  2. 클래스선택자2
  3. 클래스선택자3
  4. 클래스선택자4
