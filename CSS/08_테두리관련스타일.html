테두리 관련 스타일(border)

        선택자{
            *테두리 스타일
            border-위치-style:none/dotted/dashed/solid/double/groove/ridge/inset/outset;
        }
        선택자{
            *테두리 두께
            border-위치-width:값;
        }
        선택자{
            *테두리 색상
            border-위치-color:색상;
        }
        선택자{
            *테두리 두께,스타일,색상 한꺼번에 지정가능
            border-위치:두께값 스타일 색상; ->색상 생략시 자동으로 글자색과 통일
        }
        선택자{
            *테두리 모서리를 둥근 모양으로
            border-위/아래-좌/우-radius:숫자; -> 위치 생략시 사방이 둥글어진다.
        }
        선택자{
            *박스에 그림자 효과
            box-shadow:가로거리(x) 세로거리(y) 흐림정도 번짐정도 색상;
        }
        -위치는 위/아래/좌/우(top/bottom/left/right)가능하고 생략가능
