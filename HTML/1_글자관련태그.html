문단을 나누는 태그

문단 영역을 나누는 태그로는 p 태그와 pre 태그가 있다.
단, p 태그는 줄바꿈 입력을 별도의 태그로 지정해야 한다.
그리고 공백은 한개의 공백만을 표시하기 때문에
공백 기호(  )를 사용해야한다. 

pre 태그는 여러       공백이 인식 가능하고 
        줄 바꿈을 포함해 입력 형식 그대로를 유지하는 태그

그밖의 글자를 다루는 태그들

일반 글꼴 (태그로 감싸지 않은 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그(스크린 리더로 읽을때 강조해준다)

small : 작은 글씨

big : 큰 글씨

em: 글자를 기울여 표현하는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

strike : 글자에 취소선을 넣어주는 태그(더이상 지원하지 않음)

s : 취소선 을 넣어주는 태그

del : 취소선을 넣어주는 태그

html 태그 응용해보기

태그들은 해당 태그 내에서 중첩으로 사용 가능하다. 굵은글자를 사용할 수도 있고, 기울이거나,취소선을 넣는 등 다양하게 사용할 수있다.
형광펜을 넣어 글자를 강조할 수있다.

sub : 기본 글자에 아래첨자를 추가하는 태그

sup : 기본 글자에 윗 첨자를 추가하는 태그
