form 태그 내의 submit 버튼 클릭 시 해당 form 내에서 작성된
사용자가 입력한 값들을 서버에 전달하며 요청을 보내는 역할

form action="" method=""
-action 속성 : 해당 form내에 사용자가 입력한 값들을 전달하며 요청할 서버의 경로를 제시 -method 속성 : 해당 전송 방식을 지정하는 속성(get(기본값)/post)
>get방식 : 요청시 사용자가 입력한 값들이 url에 노출 되는 방식으로 조회의 용도로 많이 사용한다.
>post방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식으로 정보변경,정보추가 등의 용도로 사용한다
form 태그 내의 제출(submit) 버튼 클릭시 action에 지정한 서버로 method에 작성한
요청 전송 방식으로 사용자가 입력한 값들을 key&value세트로 전달한다.
